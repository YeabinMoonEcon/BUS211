What is the supervised learning?
The most common supervised learning tasks are regression (predicting values) and
classification (predicting classes)
What's the difference?
What's the example of classification problems?
  • Classification is a predictive task in which the response variable $y$ is discrete or categorical
    • discrete: it takes on a set of discrete numerical values
    • categorical: it belongs to a set of categories
      (also called classes)
  • Examples
    • Is a credit card user going to default?
    • Which product will a consumer buy?
    • Which market will a firm enter?
    • Which candidate will an individual vote for?
When this modeling frame would make sense?
To fit within this framework, the set of alternatives needs to exhibit three characteristics
  • First, the alternatives must be mutually exclusive from the decision maker's perspective
    • Choosing one alternative necessarily implies
      NOT choosing any of the other alternatives
    • The decision maker chooses only ONE option from the choice set
  • Second, the choice set must be exhaustive, in that all possible alternatives are included
    • The decision maker necessarily chooses one of the entire potential alternatives
  • Third, the number of alternatives must be finite
  • The first and second criteria are not restrictive
    • Appropriate definition assure that the alternatives are mutually exclusive and the choice set is exhaustive
    • For example, suppose two alternatives A and B are not mutually exclusive because the decision maker can choose both
    • The alternatives can be redefined to be "A only," "B only," and "both A and B"
      • opt-out option is also fine: "none of them"
Often the researcher can satisfy these two conditions in several different ways
The appropriate specification of the choice set
is governed largely by the goals of the research and
the data that are available to the researcher
Let's continue a thoughtful exercise
  • Consider households' choice among heating fuels
    • The available fuels are usually natural gas, electricity, oil, and wood
    • Do these alternatives violate both mutual exclusivity and exhaustiveness?
    • The alternatives are not mutually exclusive because a household can have two types of heating
    • The set is not exhaustive because the household can have no heating
The researcher can handle each of these issues
in several ways
  • To obtain mutually exclusive alternatives,
    one approach is to list every possible combination of heating fuels as an alternative
    • “electricity alone,” “electricity and natural gas, but no other fuels,” and so on
  • Another approach is to define the primary heating source among others
    • Under this procedure, the researcher develops
      a rule for determining which heating fuel is primary when a household uses multiple fuels
    • By definition, only one fuel (electricity, natural gas, oil, or wood) is primary
  • The advantage of listing every possible combination
    • avoids the need to define a “primary” fuel
      • a difficult and somewhat arbitrary distinction
    • also, we can examine the factors that determine households' use of multiple fuels
  • Downside is the increase in data requirment
    • eg. every cost combo of fuel
The researcher will need to take these trade-offs into consideration when specifying the choice set
The same type of issue arises
with regard to exhaustiveness
Researcher can either include “no heating” as an alternative or can redefine the choice situation as being the choice of heating fuel conditional on having heating
  • The first approach allows the researcher to examine the factors that relate to whether a household has heating
    • this ability is only realized if the researcher has data that meaningfully relate to whether or not a household has heating
  • Under the second approach, the researcher excludes from the analysis households without heating, and, by doing so, is relieved of the need for data that relate to these households
As we have just described, the conditions of mutual exclusivity and exhaustiveness can usually be satisfied, and the researcher often has several approaches for doing so
The problem is in fact on the third condition:
the number of alternatives is finite
This is in which the classfication problem would be different from regression models
Consider the savings problem:
How much money would you save?
If there is an infinite number of alternatives, classification models cannot be applied
Often regression models and classification models are distinguished by saying that regressions examine choices of how much and classification models examine choice of which
This distinction, while perhaps illustrative,
is not actually accurate
  • Consider the choice of how many cars to own
    • the alternatives are 0, 1, 2, and so on, up to the largest number that the researcher considers possible (or observes)
    • contain a finite number of mutually exclusive and exhaustive alternatives, appropriate for classification analysis
    • can also define the choice set more succinctly as 0, 1, and 2 or more vehicles
In the case of savings accounts, every one-dollar increment (or even every one-cent increment) can be considered an alternative, and as long as some finite maximum exists, then the choice set fits the criteria for classification choice
Whether to use regression or classification models in these situations is a specification issue
that the researcher must consider
